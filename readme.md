# DevCamper API

> Backend API for DevCamper application,
> which is a bootcamp directory website

# Usage

Rename "config/config.env.env" to
"config/config.env" and update the
value/setting to your own

## Install Dependencies

```
npm install
```

## Run App

```

# Run in dev mode

npm run dev


# Run in pro mode

npm start

```
